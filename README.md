# alert-orchestration-service

Plataforma serverless de ingestão multi-fonte e orquestração de eventos na AWS.

## Requisitos

- Node.js 20.x (LTS)
- npm 10+

## Scripts

- `npm run lint`
- `npm run test`
- `npm run test:coverage`
- `npm run package`

## Setup inicial

```bash
npm ci
npm run test
```
